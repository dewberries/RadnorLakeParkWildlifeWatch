Radnor Wildlife Watch

Educational park map and wildlife observation demo

Before You Explore

Please respect wildlife and follow all posted park rules while using this application. Wildlife should be observed from a safe and respectful distance, and visitors should remain on approved public trails and accessible areas whenever possible.

Some areas of the park are environmentally sensitive or restricted for habitat protection. In particular, the Eagle Protection Zone is a protected area and should not be entered. This zone is shown in the application to help communicate park stewardship and responsible use.

About This App

This application was created for educational purposes as part of a web mapping project. It is intended to demonstrate a mobile-friendly park map, interactive layers, map filtering, and user-centered design concepts.

The wildlife sightings shown in this application are mock data created for demonstration. They are included to support map interaction, filtering, and prototype functionality and should not be interpreted as official or current wildlife observations.

Data and Sources

Park features, trails, facilities, water features, and habitat-related content were assembled from public, reference, and educational sources for this project. Species reference information was informed by publicly available Radnor Lake materials and related park information.

This application is not an official park management tool and should not be used as an authoritative source for navigation, wildlife reporting, or restricted area enforcement.

About This App

Wildlife and Park Use

Please respect wildlife and follow park rules while using this application. Visitors should remain on approved public routes and avoid disturbing animals or entering sensitive habitat areas.

The Eagle Protection Zone is a restricted habitat area shown on the map to communicate wildlife protection and responsible park use. This area should not be entered by visitors.

Educational Purpose

This application was created as an educational project and is intended to demonstrate web mapping, park-oriented user experience design, interactive filtering, and spatial data visualization.

Wildlife Sightings Disclaimer

Wildlife sightings displayed in this app are mock data. They were created for demonstration purposes to support application functionality such as map display, filtering, trail-based queries, and user submission workflow examples.

Data Sources

Park layers and reference content used in this application were compiled from public, educational, and project reference materials. Species information was informed by publicly available Radnor Lake reference content and related park information.

This application is not official park guidance and should not be treated as an authoritative source for wildlife records, restricted area enforcement, or operational park decisions.
